### Required
- Node.js > 10.x
- Python > 3.x
- Mysql > 5.6

## Tech/framework used

- Next.js (Frontend)
- Flask (Backend)
- Mysql (Database)


## Installation


```sh
make install
```

## Start
```sh
make start
```

## Tests

```sh
make test
```
